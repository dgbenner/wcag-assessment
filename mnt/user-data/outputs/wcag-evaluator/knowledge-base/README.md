# Knowledge Base

This folder contains the WCAG evaluation knowledge base files.

## Current Contents

**WCAG_2_1_Accessiblity_Compliance_Checklist__Draft__Arthur_.xlsx**
- Comprehensive WCAG 2.1 criteria (78 items)
- Testing procedures for each criterion
- Role responsibilities (Design/Dev/Product)
- Helpful resources and references

## Structure

The Excel file contains three sheets:

### 1. Guidelines
- WCAG success criteria organized hierarchically
- Compliance levels (A, AA, AAA)
- Brief summaries
- Role responsibility markers

### 2. How to testQA Checklist
- Detailed testing procedures
- Screen reader testing steps
- Mobile testing methods
- Tool recommendations

### 3. Helpful Resources
- Official W3C documentation
- Platform guidelines (iOS, Android)
- Professional auditing guides
- Legal compliance references

## Usage in Claude Project

1. Upload this Excel file to your WCAG Evaluator Claude Project
2. Claude will be able to reference all three sheets
3. Ask specific questions to retrieve relevant criteria and testing procedures
